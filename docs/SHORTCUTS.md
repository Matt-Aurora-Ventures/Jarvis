# Jarvis Telegram Bot - Keyboard Shortcuts & Quick Commands

## Command Aliases

For faster access, these short aliases are available:

| Short | Full Command | Description |
|-------|--------------|-------------|
| `/a` | `/analyze` | Analyze a token |
| `/w` | `/watchlist` | View/manage watchlist |
| `/s` | `/stats` | View CLI statistics |
| `/l` | `/leaderboard` | View trading leaderboard |
| `/p` | `/portfolio` | View portfolio overview |
| `/b` | `/balance` | Check wallet balance |
| `/d` | `/dashboard` | Trading dashboard |
| `/q` | `/queue` | View CLI queue |

## Settings

### Theme Toggle
- `/theme dark` - Dark mode (default, crypto aesthetic)
- `/theme light` - Light mode (minimal emojis)

### Timezone
- `/timezone America/New_York` - Set US Eastern time
- `/timezone UTC` - Set to UTC (default)
- `/timezone` - Show current timezone and options

### Common Timezones

| Command | Zone |
|---------|------|
| `/timezone America/New_York` | US Eastern |
| `/timezone America/Los_Angeles` | US Pacific |
| `/timezone Europe/London` | UK |
| `/timezone Asia/Tokyo` | Japan |
| `/timezone UTC` | UTC |

## Search

- `/search <query>` - Search tokens by name/symbol
- `/searchp <query>` - Search with live prices

Examples:
- `/search jup` - Find Jupiter
- `/search bonk` - Find Bonk
- `/search dog` - Find all dog-themed tokens

## Export

- `/export csv` - Export trading data as CSV files
- `/export json` - Export as single JSON file
- `/exportpos` - Quick positions export
- `/exporttrades` - Quick trades export

## Tips

1. **Fastest Analysis**: Use `/a SOL` instead of `/analyze SOL`
2. **Quick Portfolio Check**: Just type `/p`
3. **Watch a Token**: Click :star: button on any token view
4. **Dark Mode**: Better for night trading, uses more emojis
5. **Light Mode**: Cleaner, fewer emojis, easier to read
