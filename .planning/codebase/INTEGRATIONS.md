# Integrations

## Confirmed External Integrations
- **Solana Network**: Executing trades, managing treasury, tracking wallets.
- **Jupiter DEX**: Used for token swaps and treasury trades.
- **CoinGecko API**: Used for backtesting data, pricing, token info.
- **Telegram API**: Heavy integration with `tg_bot` to push intel reports and take commands.
- **Twitter / X API**: Used by `twitter/autonomous_engine.py`.
- **Grok AI / xAI**: Used for parsing contracts and evaluating bonding curves/tokens.
- **Supermemory AI**: For knowledge management and contextual data retrieval.
- **NotebookLM**: For documentation query APIs and memory references.
- **Bags.fm**: Integrated system (Bags intel bot)
- **Ollama**: Local AI execution engine.
