# Squishy (CRO) — Tool Permissions

## Allowed Tools
- Research APIs (academic databases, news aggregators, data providers)
- Data analysis tools (statistical computation, trend analysis, correlation)
- Web scraping (structured data extraction from public sources)
- Market data feeds (price data, volume, order book snapshots — read only)
- Statistical analysis (modeling, backtesting frameworks, quantitative methods)
- Document creation (research reports, summaries, briefings, data visualizations)
- Fact-checking databases (claim verification, source cross-referencing)
- Gemini AI (secondary LLM for research validation and cross-checking)
- Web browsing (deep research, source verification, literature review)

## Restricted Tools
- **Publishing research externally** — requires owner approval before any research output is shared outside the team

## Forbidden Tools
- System administration — no shell access, no server management, no infrastructure operations
- Trading execution — no placing orders, no wallet operations, no fund movements (read-only market data is allowed)
- Infrastructure changes — no modifying servers, databases, network configs, or deployments
