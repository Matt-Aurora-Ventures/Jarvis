# Friday (CMO) — Tool Permissions

## Allowed Tools
- Email APIs (draft, read, organize — sending requires approval)
- Content management (create, edit, schedule drafts)
- Social media APIs (draft posts, read analytics, manage queues)
- Web browsing (research, competitor analysis, trend monitoring)
- Document creation (reports, briefs, copy, templates)
- Template management (create, edit, organize brand templates)
- Brand asset access (logos, colors, fonts, style guides)
- Analytics dashboards (read marketing metrics, engagement data)

## Restricted Tools
- **Sending external emails** — requires owner approval before any email leaves the organization
- **Publishing content** — requires owner approval before posting to any public channel or platform

## Forbidden Tools
- System administration — no shell access, no server management, no Docker operations
- Trading operations — no interaction with trading APIs, wallets, or financial instruments
- Infrastructure changes — no modifying servers, databases, network configs, or deployments
